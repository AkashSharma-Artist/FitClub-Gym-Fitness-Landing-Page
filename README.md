# FitClub – Gym & Fitness Landing Page

![](https://github.com/AkashSharma-Artist/FitClub-Gym-Fitness-Landing-Page/blob/main/ChatGPT%20Image%20Jul%209%2C%202026%2C%2005_46_27%20PM.png)

FitClub is a fully responsive single-page website for a gym/fitness brand, 
built using HTML5 and pure CSS3 with a sleek dark theme. It focuses on 
strong visual hierarchy, custom typography (Poppins), and decorative 
UI effects like blurred glow backgrounds and outlined typography.

## Features
- Hero section with bold outlined typography and CTA
- **Explore Program** section (Strength, Physical Fitness, Fat Loss, Weight Gain)
- **Class** section highlighting trainer-led sessions
- **Why Join Us** section with overlapping image + feature grid
- **Pricing Plans** section (Basic, Weekly, Monthly tiers)
- **Member Reviews** section with ratings
- Fully responsive design (desktop, tablet, mobile breakpoints)
- Custom dark theme using CSS variables, RemixIcon icon library, and Google Fonts (Poppins)

## Tech Stack
- HTML5
- CSS3 (CSS variables, Grid, Flexbox, media queries)
- RemixIcon
- Google Fonts
